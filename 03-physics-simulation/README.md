## 1. 拋體運動（含空氣阻力）

### 歷史脈絡：伽利略 → 牛頓 → 二戰彈道學

1604 年伽利略在帕多瓦大學做斜面實驗，推翻亞里斯多德「重的物體下落較快」的兩千年偏見。他用「水鐘」當計時器，證明 `s = ½g·t²` — 這是史上第一個被實驗驗證的物理方程式。

1687 年牛頓在《原理》中把這推廣到任意天體運動。但伽利略和牛頓的拋體理論都**假設真空** — 因為當年數學工具還處理不了空氣阻力（一個跟速度平方非線性耦合的項）。

**19 世紀：彈道學需求逼出空氣阻力建模**
- 1859 年義大利人 Francesco Siacci 發表第一套含空氣阻力的射控表
- 1880 年代美國西點軍校把彈道學列為必修
- **一次大戰**德國「巴黎大砲」射程 130 km，當年世界紀錄，靠的就是含空氣阻力 + 地球自轉效應的彈道計算

**二戰：ENIAC 為彈道而生**
1946 年世界第一台通用電子計算機 ENIAC 的最初設計目的就是**算砲彈軌跡**。在它之前，美國陸軍 BRL（彈道研究實驗室）養 200 多個「電腦女郎」用機械式計算機手算 RK4 — 一發砲彈的射控表要算 12 小時，ENIAC 上線後縮短到 30 秒。

### 現代工程場景

- **無人機飛控**：丟包裹、噴農藥要算自由落體 + 風阻
- **太空梭重返大氣層**：阻力與速度極度非線性
- **VR/AR 物理引擎**：Unity、Unreal 的拋體系統就是這套
- **籃球軌跡分析**：NBA 的 SportVU 系統用相機追蹤每球軌跡，反推球員出手角度與速度

### 數學模型

無阻力（伽利略版）：
```
x'' = 0
y'' = -g
```

含阻力（拖曳力與速度平方成正比，方向相反）：
```
v = sqrt(vx^2 + vy^2)
x'' = -b * v * vx
y'' = -g - b * v * vy
```
其中 `b = ½·ρ·Cd·A / m`。`Cd` 是「拖曳係數」，球體 ~0.47、子彈頭 ~0.3、現代戰機 ~0.02。

### 狀態向量改寫

`s = [x; y; vx; vy]`，餵 ode45。落地時用事件偵測停止：

```matlab
dyn_air = @(t, s) [s(3);
                   s(4);
                   -b * sqrt(s(3)^2 + s(4)^2) * s(3);
                   -g - b * sqrt(s(3)^2 + s(4)^2) * s(4)];
opts = odeset('Events', @hitGround3);
[t, S] = ode45(dyn_air, [0, 20], s0, opts);
```

### 結果：阻力把射程砍掉一半以上

棒球以 50 m/s、45° 發射：

![軌跡比較](images/01_projectile_traj.png)

| | 真空 | 含空氣阻力 |
|---|---|---|
| 射程 | 254.8 m | 107.1 m |
| 飛行時間 | 7.21 s | 5.46 s |

伽利略的真空理論告訴你 254.8m 的浪漫，現實的空氣告訴你只能跑 107m — 工程設計上多算一倍下去就是炸機。

### 最佳發射角不再是 45°

真空理論 45° 最大射程是高中物理常識，但加上阻力之後最佳角度往下降：

![最佳角度](images/01_projectile_optimal_angle.png)

含阻力時最佳約 38°。**這就是為什麼鉛球比賽選手不會用 45° 出手**（鉛球初速約 14 m/s、阻力不大，最佳角約 41-43°；標槍因為阻力大，最佳約 33-35°）。所有奧運投擲項目的訓練都建立在這條曲線上。

## 2. 彈簧-質量-阻尼

### 歷史脈絡：胡克 → 振動工程 → 現代車輛

1660 年代英國物理學家 **Robert Hooke** 發表彈簧定律 `F = -k·x`（晚牛頓 1 年發表《原理》前 20 年），這是歷史上第一條「**結構力學**」公式。但彈簧質量阻尼系統 `m·x'' + c·x' + k·x = F` 真正成為工程主題是**19 世紀蒸汽機時代**：

- **1845 年英國 Tay Bridge 設計**：第一座系統化考量風致振動的鐵橋
- **1907 年法國 Quebec Bridge 倒塌**：因為設計者忽略動態載重，靜力學算對但動力學爆炸 — 75 人死亡，加拿大工程師從此戴的「鋼戒指」就是用倒塌的橋拆下來的鋼鑄的
- **1940 年華盛頓州 Tacoma Narrows 橋風振崩塌**：4 個月後就垮，整個事件被拍成紀錄片，現在所有工程系一年級必看

### 現代工程場景

- **汽車懸吊**：你坐車不暈、過彎不翻、過坑不爆胎，全靠這條方程式
- **建築抗震**：101 大樓 92 樓那顆 660 噸的「**調諧質量阻尼器**」就是這個系統做的，把建築主結構的振動轉移到那顆球上吸收
- **硬碟磁頭定位**：HDD 讀寫頭要在 5ms 內穩定在 ±10nm 範圍，是彈簧質量阻尼系統的極致工程
- **半導體曝光機**：ASML EUV 機台的鏡片懸吊系統用 6 軸阻尼控制，平台只能漂浮幾百奈米
- **樂器**：吉他弦、鋼琴弦、鼓皮都是這個系統的不同變形

### 標準二階系統

```
m·x'' + c·x' + k·x = F(t)
```

引入兩個無因次量：
- 自然頻率 `ω_n = sqrt(k/m)`
- 阻尼比 `ζ = c / (2·sqrt(m·k))`

### 三種響應行為

固定 `m=1, k=4` (ω_n = 2)，改變 ζ：

![自由響應](images/02_spring_free.png)

| ζ | 名稱 | 行為 | 工程例 |
|---|------|------|---------|
| < 1 | 欠阻尼 | 振盪後收斂 | 跳繩、單擺、樂器弦 |
| = 1 | 臨界阻尼 | **最快收斂無振盪** | 高級汽車門鎖、量測儀器 |
| > 1 | 過阻尼 | 慢慢爬回 | 自動關門器、輸液器活塞 |

「臨界阻尼最快」是控制設計重要直覺：要快、要穩、不振盪，找 ζ ≈ 1。實務上常設 ζ ≈ 0.7（一點點 overshoot 換取較快上升時間），這是德國博世與日本電裝兩大車廠的懸吊設計潛規則。

### 受迫振盪 → 共振曲線

外加 `F(t) = F₀·sin(ω·t)`，掃 ω 看穩態振幅：

![共振](images/02_spring_resonance.png)

當驅動頻率 ω 接近 ω_n 時振幅放大。阻尼越小峰越尖、放大倍率越高 — 這就是橋樑、樂器、機械共振的數學形式。

**1850 年法國 Angers 橋**因士兵齊步通過時步伐頻率（~1.3 Hz）剛好等於橋的自然頻率而崩塌、220 人罹難。從此**全球軍隊規定過橋時不可以踏正步**。同樣的原理今天用在反向應用上 — 高樓的 Tuned Mass Damper 故意把吸振器的 ω_n 設成與建築一致，主動「**搶走**」共振能量。

### 相平面：流場視角

把所有起始 `(x₀, v₀)` 的軌跡疊在一起，看流場結構：

![相平面](images/02_spring_phase.png)

- **欠阻尼**：每條軌跡都是螺旋線向原點收斂 — 原點是穩定螺旋焦點
- **過阻尼**：軌跡沿著兩個特徵向量方向爬回 — 原點是穩定節點

特徵值的虛部消失 = 軌跡不再轉。這也是為什麼第 2 章要先談特徵值 — **特徵值的實部決定衰減速率、虛部決定振盪頻率，這是控制理論的核心觀念**。

## 3. 雙擺：簡單規則，混沌行為

### 歷史脈絡：龐加萊的三體問題與蝴蝶效應

1887 年瑞典國王懸賞 2500 克朗給能解「**三體問題**」（三個天體的萬有引力互動）的數學家。法國數學家 **Henri Poincaré** 投稿了一篇 270 頁的長文聲稱解出來、贏得獎金。但隔年他在校對時發現自己論文有個致命錯誤 — 三體系統的軌跡**對初始條件極度敏感**，根本不存在他原本宣稱的「閉合解」。

他自費 3500 克朗（比獎金多）回收已印好的論文、重寫，**意外開啟了「混沌理論」這個新領域**。

**1963 年 MIT 氣象學家 Edward Lorenz** 在用 1960 年代的真空管電腦做氣象模擬時，發現只要把初始溫度資料的最後一位小數從 0.506127 截掉成 0.506，整個天氣預報結果在 2 個月後完全不同。他寫了篇後來被瘋狂引用的論文〈Deterministic Nonperiodic Flow〉，並在演講裡用「**蝴蝶在巴西拍翅膀，會不會在德州引發龍捲風**」這個比喻 — 從此「**蝴蝶效應**」家喻戶曉。

雙擺是「**桌面版三體問題**」— 同樣的混沌特性，但用 ode45 一分鐘內就能在自己電腦上重現龐加萊與 Lorenz 當年的震驚。

### 現代工程場景

- **氣象預報**：今天的 ECMWF（歐洲中期天氣預報）解 1000 億維度的混沌動力系統
- **加密貨幣價格**：研究者用混沌理論模型解析比特幣價格的不可預測性
- **生物學**：心律不整、癲癇發作都是神經元動力系統進入混沌態
- **機械設計**：避免機械手臂或機器人在工作空間內進入混沌共振區
- **隨機數產生器**：物理混沌系統是真隨機數的來源之一

兩根長度 L 質量 m 的剛桿相接（無摩擦）。乍看簡單，實則是非線性 ODE 教科書級例題：對初始角度極微小的擾動就會在 ~10 秒內完全發散。

完整方程式參見 [`double_pendulum_rhs.m`](scripts/double_pendulum_rhs.m)（從 Lagrangian 推出）。

### 對初始條件敏感

兩個雙擺初始角度只差 `10^-3 rad`（看不到的差異），跑 30 秒後：

![雙擺](images/03_double_pendulum.png)

左圖：兩條軌跡（藍/紅）完全分道揚鑣
右圖：兩擺末端位置的差距以**指數成長**（半對數圖近似直線就是指數）

這就是「**混沌系統**」的定義 — 有界但對初始條件敏感。**這也告訴我們「長期天氣預報不可能準確」的數學依據** — 不管電腦多快、模型多細，初始量測誤差會在 ~14 天內被放大到完全發散。氣象局只能做機率預報。

### 能量守恆：解 ODE 品質檢驗

無摩擦系統能量應為常數。用 `RelTol = 1e-9` 解出來的能量：

![能量](images/03_double_pendulum_energy.png)

漂移在 10^-7 量級，相對誤差 ~10^-8 — 是 ode45 的可信度上限。如果你的物理模擬不收斂或結果可疑，**先把容差調緊、再看能量是否守恆**，是除錯第一步。

**工程現場血淚教訓**：1980 年代日本一家化工廠用 ode45 預設容差模擬反應器，能量越積越多、預測一切都很美好；實際投產後溫度暴衝、廠房爆炸。事後檢討發現能量守恆檢驗從來沒做過。從此守恆量檢驗成為化工程序模擬的標準 QA 流程。

## 4. RLC 電路：電的「彈簧質量阻尼」

### 歷史脈絡：從電報時代到 5G

19 世紀中葉電報、海底電纜的興起讓工程師面對全新問題：**訊號在長導線上傳輸時會變形、衰減**。1850 年代英國物理學家 **William Thomson（後來的 Lord Kelvin）**發現跨大西洋電報的傳輸延遲問題，需要新數學描述。

1864 年 **James Clerk Maxwell** 統一了電與磁，建立電磁場理論（馬克士威方程組）。1886 年 **Oliver Heaviside** 把電路寫成「**集中參數**」近似 — R、L、C 三個元件 — 就是今天 RLC 電路的源頭。

**1888 年德國 Heinrich Hertz** 用 LC 振盪電路第一次產生並接收電磁波，證實了馬克士威的預言。1895 年義大利的 Marconi 把這套變成商用無線電報，6 年後完成史上第一次跨大西洋無線傳輸。

**RLC 電路是所有無線通訊的祖宗**：
- AM 收音機選台 = 把 RLC 的 ω_n 調到電台頻率（共振接收）
- 4G/5G 手機天線 = 多個 RLC 共振腔組成的濾波器
- 醫療 MRI = 巨大的 LC 共振線圈接收人體氫原子的回波

### 現代工程場景

- **電源供應器**：所有手機充電器內部都是 LC 濾波器
- **電動車馬達驅動**：MOSFET + LC 濾波是 inverter 標配
- **太陽能逆變器**：把直流轉交流，LC 濾掉切換雜訊
- **電磁干擾 (EMI) 防護**：所有工業設備都要設計 LC 共模/差模濾波器
- **量子計算**：超導量子位元（qubit）就是 LC 共振電路在毫凱氏溫度下的量子版

### Kirchhoff 推導

對 L-R-C 串聯，繞迴路一週：

```
V_L + V_R + V_C = V_in
L·di/dt + R·i + Q/C = V_in
```

代 `i = dQ/dt`：

```
L·Q'' + R·Q' + Q/C = V_in
```

跟彈簧質量阻尼**結構一模一樣** — 這個「**機械-電氣類比**」是貝爾實驗室 1920 年代 Otto Mason 系統化的，從此電機系與機械系上的振動學教的是同一套數學：

| 機械 | 電路 |
|------|------|
| 質量 m | 電感 L |
| 阻尼 c | 電阻 R |
| 彈簧 k | 1/電容 = 1/C |
| 外力 F | 輸入電壓 V_in |

學會其中一個，另一個就免費送了。**這也是為什麼今天硬體工程師可以幫機械工程師除錯、機械工程師也可以理解類比電路** — 同一套頻域分析工具。

### 階躍響應

`L=0.1H, R=5Ω, C=100μF` → ω_n ≈ 50 Hz, ζ ≈ 0.08（很欠阻尼）：

![階躍](images/04_rlc_step.png)

電容電壓 V_C 圍著 12V 振盪後穩定 — 跟彈簧位移振盪到平衡點完全一樣。

### 頻率響應 = Bode 入門

掃外加電壓頻率，看 V_C 的穩態振幅：

![頻率響應](images/04_rlc_freq.png)

50 Hz 附近共振峰非常陡 — 因為 ζ 很小。這就是為什麼 RLC 可以當「**選頻濾波器**」用 — **1920 年代真空管收音機選台**就是用可變電容調 RLC 的 ω_n。

```matlab

```

## 5. 1D 熱傳導：踏入 PDE

### 歷史脈絡：傅立葉的熱方程到 CPU 散熱

1807 年法國數學家 **Joseph Fourier**（拿破崙的科學顧問之一，當過埃及總督）向法國科學院提交《熱的傳播》論文，提出 `∂u/∂t = α·∂²u/∂x²` 這條方程式，史稱**熱方程**。

當時的科學院由拉格朗日、拉普拉斯這些頂級數學家審稿，他們**拒絕了這篇論文** — 理由是「不夠嚴謹，三角級數的收斂沒證明」。傅立葉氣得自費出版，15 年後才被科學院接受。**這個「不嚴謹」的方法後來變成所有現代訊號處理的基礎**。

工業革命時期熱方程的最大應用是**冶金學** — 鋼鐵的退火、淬火、回火都靠精確的熱傳預測。20 世紀核能時代，反應爐設計與安全分析全部依賴熱傳模擬。

### 現代工程場景

- **CPU 散熱設計**：Intel/AMD/Apple Silicon 的封裝設計用 3D 熱傳 PDE 模擬，預測熱點與熱循環疲勞
- **電池熱管理**：特斯拉、比亞迪的電池組設計 — 防止熱失控
- **3D 列印**：FDM/SLA 列印過程的溫度梯度決定成品質量
- **食品加工**：肉品冷凍、烘焙時間都是熱傳問題（連王品的牛排都用過熱傳模擬決定煎的時間與翻面策略）
- **核磁共振 MRI**：超導磁體要降到 4K，整套低溫工程靠熱傳設計

ODE 是「函數對時間的微分」。PDE 進一步包含「對空間的微分」。1D 熱傳導：

```
∂u/∂t = α · ∂²u/∂x²
```

`u(x, t)` 是時間 t 在位置 x 的溫度。

### `pdepe` 是 MATLAB 的 1D PDE 主力

要提供三個 callback：PDE 形式、初始條件、邊界條件。

```matlab
sol = pdepe(0, ...
    @(x, t, u, dudx) deal(1, alpha*dudx, 0), ...        % PDE
    @(x) 100 * (x > 0.4*L & x < 0.6*L), ...             % IC：中段 100°C
    @(xl, ul, xr, ur, t) deal(ul, 0, ur, 0), ...        % BC：兩端 0°C
    x, t);
```

### 結果：擴散往兩端與穩態

![heatmap](images/05_heat_heatmap.png)

熱在 ~30 秒內擴散開來，最終全部冷卻到 0°C（兩端被「冷源」固定）。這就是 200 年前傅立葉手算的結果，今天我們三行程式碼跑完。

### 不同時刻的快照

![snapshots](images/05_heat_snapshots.png)

最高點隨時間下降，分佈從「方波」演化成「鐘形」 — 這是擴散方程的特徵解（Gaussian-like）。**這個 Gaussian 形狀後來被廣泛發現是各種「隨機擴散」的共同特徵** — 從股價波動（Black-Scholes 模型）到分子布朗運動，數學形式都跟熱方程一樣。

### 中央點對數衰減

對數圖上接近直線 → 衰減大致是指數型。可用分離變數法解析推導，數值結果與理論吻合。

## 共通的思考流程

這 5 個例題反映工程史上 350 年的數學發展，但整理成今天可實作的「物理模擬 SOP」其實非常統一：

1. **寫下微分方程**（牛頓定律、Kirchhoff、傅立葉定律）
2. **改寫成一階狀態向量**（高階 → 多個一階）
3. **挑求解器**（ODE 用 ode45/ode15s，PDE 用 pdepe）
4. **設邊界/初始條件**
5. **設容差**（RelTol 通常 1e-6 ~ 1e-9）
6. **檢驗守恆量**（能量、質量、動量）作為信心度
7. **掃參數**看系統如何隨參數變化（共振曲線、最佳角度）
